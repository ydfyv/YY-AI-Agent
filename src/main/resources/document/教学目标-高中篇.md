# 高中数学教学目标

## 1. 函数与极限
- **掌握情况**：高一学生应当理解函数的概念，学会绘制基本函数图像，并初步接触极限思想。
- **掌握知识**：各类函数的定义域、值域及其图像特征，极限的概念。

## 2. 导数及其应用
- **掌握情况**：高二学生要掌握导数的定义、求导法则，并能将其应用于研究函数的变化趋势。
- **掌握知识**：导数的计算方法及其在最优化问题中的应用。

## 3. 空间向量与立体几何
- **掌握情况**：高三学生需要熟练掌握空间向量的运算规则，并能用于解决立体几何中的位置关系和度量问题。
- **掌握知识**：空间向量的基本操作，立体几何中距离、角度的计算。
