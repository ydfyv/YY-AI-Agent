# 小学数学教学目标

## 1. 数的认识与运算
- **掌握情况**：学生在学习完小学一年级时，应该能够认识并书写0到100以内的数字，并进行简单的加减法计算。
- **掌握知识**：数的概念、基本的加减法运算。

## 2. 几何图形的认知
- **掌握情况**：到了三年级，学生应能识别并分类常见的平面图形（如圆形、正方形、长方形）和立体图形（如球体、立方体）。
- **掌握知识**：常见几何图形的特点及其名称。

## 3. 分数与小数的理解
- **掌握情况**：五年级结束时，学生需要理解分数和小数的意义，能够在实际问题中应用。
- **掌握知识**：分数和小数的基本概念及简单运算。
